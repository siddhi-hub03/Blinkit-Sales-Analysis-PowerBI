# Blinkit-Sales-Analysis-PowerBI

Project Overview

This project presents an interactive Power BI dashboard for analyzing Blinkit's sales performance across various outlet types, locations, item categories, and outlet sizes.

The dashboard helps identify key business trends and supports data-driven decision-making.

Objectives:
Analyze total sales performance.
Compare sales across outlet locations.
Study item category performance.
Evaluate outlet size contribution.
Monitor average ratings and item counts.
Tools Used:
Power BI
Microsoft Excel
DAX
Power Query

Dashboard Features-

Sales Analysis:
-Total sales by outlet establishment year.
-Sales contribution by outlet location.
-Sales distribution by outlet size.
Product Analysis:
-Item type analysis.
-Fat content analysis.
-Product category comparison.
Outlet Analysis:
-Outlet type comparison.
-Outlet size comparison.
-Outlet location comparison.
